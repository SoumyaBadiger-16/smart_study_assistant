# 📚 Smart Study Assistant

An AI-powered **Smart Study Assistant** built using **Google Gemini API** and **Gradio** that helps students understand concepts in an interactive and personalized way. The application allows users to choose different AI teaching personalities and receive high-quality explanations tailored to their learning style.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/SoumyaBadiger-16/smart_study_assistant.git
```

Move into the project directory

```bash
cd smart_study_assistant
```

Install dependencies

```bash
pip install -r requirements.txt
```

Create an environment variable

```
GEMINI_API_KEY=YOUR_API_KEY
```

Run the application

```bash
python app.py
```

# 🚀 Usage

1. Enter your study question.
2. Choose a personality:

   * 😊 Friendly
   * 🎓 Academic
3. Click **Submit**.
4. Receive an AI-generated explanation.
